# PKTrace

## Environment variables (`.env`)

- Create `.env` file in the project root. 

```env
MAPBOX_TOKEN="pk.xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
STREET_VIEW_KEY="MLY|xxxxxxxxxxxxxxx|xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
```

## Running the project

**1. Install dependencies**

```bash
yarn install
```

**2. Build in watch mode** 

```bash
yarn watch
```

**3. start dev server** 

```bash
yarn start:dev
```
